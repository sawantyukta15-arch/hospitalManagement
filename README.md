# Patient Management System
A simple console-based Patient Management System built using Python and Microsoft SQL Server.  
This project allows patients to register their details and enables employees to view, sort, filter, and search patient records stored in a database.
The goal of this project is to practice Python programming, database connectivity, and basic software design principles.

# Patient Module
- Add a new patient record
- Store patient details such as age, weight, condition, and insurance information

# Employee Module
- View all patient records
- Sort patients by age or weight (ascending or descending)
- Filter patients within a given age or weight range
- Search for a patient by full name

# Tech Stack
- Python
- Microsoft SQL Server
- pyodbc (ODBC connector for SQL Server)
